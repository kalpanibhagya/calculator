# Calculator

A small example project with a web UI and a Python script.

### Project structure
- `index.html` — simple web interface for the calculator.
- `main.py` — Python script (backend or utility script).

### Prerequisites
- Python 3.8+ 
- FastAPI
- Uvicorn

Run `pip install fastapi uvicorn` to install FastAPI and Uvicorn

### How to run

Run `uvicorn main:app --reload`
Open http://localhost:8000 in your browser
Now you can test this simple calculator
